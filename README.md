# 🗄 KZ Business Database

База данных казахстанских компаний с AI Smart Parser.

---

## 🚀 Запуск (3 шага)

### 1. Получите API ключ Anthropic
Зайдите на https://console.anthropic.com → API Keys → Create Key

### 2. Вставьте ключ в `.env`
Откройте файл `.env` и замените `sk-ant-PASTE_YOUR_KEY_HERE` на ваш реальный ключ:
```
ANTHROPIC_API_KEY=sk-ant-api03-...ваш ключ...
```

### 3. Запустите сервер
**Windows** — двойной клик на `start.bat`  
**Mac/Linux** — в терминале:
```bash
python3 server.py
```

Затем откройте в браузере: **http://localhost:8000**

---

## 📋 Требования
- Python 3.8+
- Интернет соединение

Зависимости установятся автоматически при первом запуске через `start.bat`.  
Или вручную: `pip install anthropic`
